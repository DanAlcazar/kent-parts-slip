Setting up your Google Sheet

  1. Go to sheets.google.com and create a new spreadsheet
  2. Name it Kent Parts Catalog
  3. Set up the first row as headers in this exact order:
    Part Number  |  Description  |  Make  |  Cost
  4. Add your parts data — one part per row. Cost = the raw cost (the form applies 50% markup automatically)
  5. Click File → Share → Publish to web
  6. Under "Link", select the sheet tab and choose Comma-separated values (.csv)
  7. Click Publish and copy the URL
  8. Paste that URL into the sheet URL field above and click Load Sheet

Example CSV URL format:

https://docs.google.com/spreadsheets/d/1ABC...xyz/pub?output=csv
